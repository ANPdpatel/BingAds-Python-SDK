Bases: bingads.v11.internal.bulk.entities.single_record_bulk_entity._SingleRecordBulkEntity

Represents a sitelink.

This class exposes the <see cref=”BulkSiteLink.SiteLink”/> property that can be read and written as fields of the Sitelink Ad Extension record in a bulk file.

For more information, see Sitelink Ad Extension at https://go.microsoft.com/fwlink/?linkid=846127.

The Sitelink Ad Extension record includes the distinct properties of the BulkSiteLink class, combined with the common properties of the BulkSiteLinkAdExtension class.

One BulkSiteLinkAdExtension has one or more BulkSiteLink. BulkSiteLink instance corresponds to one Sitelink Ad Extension record in the bulk file. If you upload a BulkSiteLinkAdExtension, then you are effectively replacing any existing site links for the sitelink ad extension.
