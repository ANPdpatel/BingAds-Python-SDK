class bingads.bulk.entities.ad_extensions.bulk_product_ad_extensions.BulkProductAdExtension(account_id=None, product_ad_extension=None, product_collection=None, identifier=None)[source]¶

Bases: bingads.internal.bulk.entities.multi_record_bulk_entity._MultiRecordBulkEntity

Represents a product ad extension.

This class exposes the product_ad_extension property that can be read and written as fields of the Product Ad Extension record in a bulk file.

For more information, see Product Ad Extension at http://go.microsoft.com/fwlink/?LinkID=511516.

The Product Ad Extension record includes the distinct properties of the BulkProductConditionCollection class, combined with the common properties of the BulkProductAdExtension class, for example account_id and ProductAdExtension.

One BulkProductAdExtension has one or more BulkProductConditionCollection. Each BulkProductConditionCollection instance corresponds to one Product Ad Extension record in the bulk file. If you upload a BulkProductAdExtension, then you are effectively replacing any existing product conditions for the product ad extension.
